Types of Evaluation

Empirical Evaluation vs. Analytical Evaluation

Empirical Evaluation (With Actual Users)

Involves real users interacting with the visualization

Collects user-centered feedback on comprehension, usability, and experience
Methods include user studies, interviews, think-alouds, surveys, and A/B tests
Captures qualitative (opinions, emotions) and/or quantitative (task time, accuracy) data
Helps reveal unexpected issues and insights about user behavior

Analytical Evaluation (Without Actual Users)

Conducted without involving users

Relies on design heuristics and or automated tools

Methods include heuristic evaluation, theoretical analysis, and accessibility checks
Fast, low-cost

Cannot fully assess user experience or story comprehension

Common Methods for Visualization Evaluations

Think-aloud protocol — Users narrate their thoughts while interacting
User surveys — Collect ratings or open-ended feedback

A/B testing — Compare two design versions for effectiveness

Task performance — Measure time, accuracy, or completion rates
Interviews & focus groups — Gather in-depth user insights

Heuristic evaluation — Experts check against design principles
Design checklists — Verify layout, color use, accessibility, etc.
Interaction logging — Analyze user behavior from usage data
Automated tools — Check color contrast, readability, etc.
